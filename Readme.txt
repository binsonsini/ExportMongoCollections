Read multiple collection names from text file and and convert it to JSON file
-----------------------------------------------------------------------------

Collection_names.txt 
Contains name of collections to be converted to json
-------------------------------------------------------
db.config
Contains name of DB  which the collection belongs to 
-------------------------------------------------------
Make sure mongoexport.exe is available in your bin folder of mongoDB installation
if not download it from the link below and copy the file to the folder
https://www.mongodb.com/try/download/database-tools
------------------------------------------------------
After setting the above files run shell script file by double clicking
